#Bongo Cat Keyboard.
A 3x3 keyboard with rotary Encoder and OLED display running bongo cat animation.
## Features
-3x3 key matrix
-Rotary Encoder with volume control 
-SSD1306 OLED display with animated bongo cat
## Setup
bash
qmk compile -kb bongo_keyboard -km default
qmk flash -kb bongo_keyboard -km default
